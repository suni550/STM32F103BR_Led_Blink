Imt.Base.HAL.STM32F103MD is the HAL implementation for the ST STM32F103x8 and STM32F103xB Cortex-M3 microcontroller.
MD = Medium-density performance line ARM-based 32-bit MCU, which is supported by STM32F103x8 and STM32F103xB devices.
